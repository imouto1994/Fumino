# Fumino

PWA app for personal manga database to keep track of my collection of purchased mangas and those to be bought in the future. ðŸ™Œ

Powered by **ReactJS**, **react-virtualized**, **cheerio**, **Typescript**, **Webpack**, **BabelJS**, **Netlify**, **Netlify Functions** & **Netlify CMS**

## Install

```bash
$ yarn install
$ yarn global add netlify-cli
$ netlify init
```

## Development

### Web App

```bash
$ yarn dev-web
```

### Admin CMS

```bash
$ yarn dev-admin
```

### Functions

```bash
$ netlify dev
```
